Current JobMine Search

Imagine you were a .NET developer who wanted to apply to all jobs that require .NET knowledge. How would you go about finding all .NET jobs? Many job titles do not contain the words “.NET” and clicking on every job in Job Inquiry is likely not ideal (learned this the hard way, it was not pretty)

JobMine EZSearch

We shouldn’t have to spend so much time looking for jobs when that time can be invested in writing applications for those jobs. Using JobMine EZSearch, we can easily find all jobs that require .NET knowledge by typing in “.NET” in the keywords search bar.

 
 

 

Why Use JobMine EZSearch?




Refined Search Mechanism
In addition to almost all of JobMine’s search filters, JobMine EZSearch can also search by keywords, prioritized languages, or you can write your own SQL queries.




Integrates with Short List
Add or remove jobs from your shortlist with a single click. A text file is also created showing all jobs you have added to shortlist.




Open to Extension
Stores jobs from JobMine to a local database so that you can use the data to build your own applications.
 



 


Fast and Functional
Once JobMine EZSearch has prepared the database, you can instantaneously* search through 3000+ jobs and view jobs without waiting for page loads




Download Formats
Download all jobs in four file formats: .html, .csv, .json, and .txt

Q&A

How do I create my own SQL queries?

SQL Search automatically selects all columns from the table storing the jobs and you can simply type in the WHERE clause to find your queries (eg, location = ‘Toronto’ OR summary LIKE ‘%game%’). You can also press the ? button next to the SQL Search textbox for help on writing SQL queries

What can I do with the database?

The database contains a table with 14 columns that store information related to JobMine jobs. You can use the data and run analytics, display the jobs in different formats, find patterns in text, or even read through them before going to bed. Liberty exists.
